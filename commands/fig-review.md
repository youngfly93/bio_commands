批量审查项目中的图表，检查是否符合 SCI 发表标准。

## 扫描范围

查找以下目录中的图片文件：
- `figures/`, `results/`, `output/`, `plots/`
- 支持格式: `.png`, `.jpg`, `.jpeg`, `.tiff`, `.tif`, `.pdf`, `.svg`

```bash
find . -type f \( -name "*.png" -o -name "*.jpg" -o -name "*.jpeg" -o -name "*.tiff" -o -name "*.tif" -o -name "*.pdf" -o -name "*.svg" \) | grep -iE "(figure|result|output|plot|fig)" | sort
```

## 审查标准

对每张图逐一检查：

### 1. 基本规范
- 背景是否为纯白（非灰色/透明）
- 是否有多余的网格线
- 分辨率是否 >= 300 DPI（检查像素尺寸）

### 2. 配色与可读性
- 配色是否协调、可区分
- 是否适合色盲读者（避免纯红绿对比）
- 文字大小是否可读（不会因缩放变得模糊）

### 3. 标签与标注
- 坐标轴是否有标签和单位
- 图例是否完整
- 标题/编号是否准确
- 统计显著性标注是否正确（*, **, ***, ns）

### 4. 数据展示
- 柱状图是否有误差线
- 箱线图是否显示了离群值
- 热图是否有色标和聚类树
- 散点图是否标注了关键点

### 5. 与数据一致性
- 图表中的数值是否与结果文件对应
- 样本数/组别是否与实验设计一致

## 注意事项

- 使用 Read 工具查看图片进行视觉检查
- 用 `identify` 或 `python3 PIL` 检查技术参数（DPI、尺寸）
- 如果图表由 R/Python 脚本生成，同时审查脚本中的绘图参数
